Computed­KeyFrames

Same easing applied during whole animation.

Both the items has the same easing easeOutQuart in this example.

The red and black has the same 5 keyframes. The easing for the black is applied during whole animation and not on each keyframe as seen with the red.

The easing values are calculated in JS and then inserted in to the DOM using style element. translate3d and scale3d are supported.
